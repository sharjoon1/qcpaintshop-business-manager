1. Overview

The WhatsApp Marketing system lets you send bulk promotional messages to your leads through WhatsApp. It uses the branch WhatsApp sessions (connected via QR code) to send messages with built-in anti-block protection.

Key Features:

  • Send text messages, images, and documents to leads in bulk
  • Select specific contacts or filter by status, source, city, etc.
  • Anti-block protection: random delays, spin text, unique messages
  • Message templates for quick campaign creation
  • Real-time progress tracking and analytics
  • Warm-up mode for new WhatsApp numbers
Important: Make sure your branch WhatsApp is connected before starting a campaign. Go to Zoho Books → WhatsApp tab to check connection status.

2. Getting Started

Prerequisites

  1. WhatsApp Connected: At least one branch must have WhatsApp connected (Zoho → WhatsApp tab)
  2. Leads with Phone Numbers: Your leads must have phone numbers saved
  3. Admin Access: Only admin users can access WA Marketing

Accessing WA Marketing

Navigate to Marketing → WA Marketing in the sidebar. The page has 5 tabs:

3. Creating a Campaign

Creating a campaign is done through a 5-step wizard:

1 Basic Info

  • Campaign Name: Give your campaign a descriptive name (e.g., "Diwali Paint Sale")
  • Description: Optional notes about the campaign
  • Branch: Select which branch's WhatsApp number to send from
  • Message Type: Choose Text, Image, or Document

2 Message Composer

Write your message using the composer. You can:

  • Use Templates: Pick a saved template from the dropdown
  • Insert Variables: Click variable buttons to insert {name}, {company}, {city} etc. These will be replaced with each lead's actual data
  • Use Spin Text: Write [Hi|Hello|Hey] — the system will randomly pick one word for each message, making every message unique
  • Preview: Click "Generate Preview" to see how the message looks with sample data

3 Select Contacts

Browse and select which leads to send the message to:

  • Search: Type to search by name, phone, company, or city
  • Filter: Use Status, Source, and Priority filters to narrow down contacts
  • Select Individual: Click on any row or tick the checkbox to select that contact
  • Select All on Page: Tick the header checkbox to select all visible contacts
  • Select ALL Matching: Click this button to select ALL contacts matching your current filters (across all pages)
Tip: Only leads with phone numbers are shown. If a lead doesn't appear, make sure their phone number is saved in the Leads module.

4 Anti-Block Settings

Configure protection against WhatsApp blocking:

  • Min/Max Delay: Random wait time between messages (default: 30-90 seconds)
  • Hourly Limit: Maximum messages per hour (default: 30)
  • Daily Limit: Maximum messages per day (default: 200)
  • Warm-Up Mode: Gradually increases the daily limit over 5 days for new numbers

5 Review & Start

Review all your campaign settings and click "Create & Start" to begin sending.

5. Anti-Block Protection

The system has multiple layers of protection to prevent WhatsApp from blocking your number:

FeatureHow It Works
Random DelaysWaits 30-90 seconds between messages (configurable)
Spin Text[Hi|Hello|Hey] → randomly picks one, making each message different
Variable Substitution{name} → each lead gets personalized message
Invisible MarkersAdds invisible characters so no two messages are identical
Rate LimitingEnforces hourly and daily sending limits
Warm-Up ModeDay 1: 20 msgs → Day 2: 50 → Day 3: 100 → Day 4: 150 → Day 5+: 200
Auto-PausePauses campaign after 3 consecutive failures
Warning: Even with all protections, sending too many messages too fast can lead to a WhatsApp ban. Use warm-up mode for new numbers and keep daily limits reasonable (under 200/day).

7. Dashboard & Analytics

The Dashboard tab shows:

  • Total Sent: All-time messages sent across all campaigns
  • Today Sent: Messages sent today
  • Active Campaigns: Currently running campaigns
  • Failed: Total failed messages
  • Campaign Performance Chart: Bar chart comparing sent vs failed per campaign
  • Hourly Volume Chart: Line chart showing today's sending pattern by hour
  • Engine Status: Green indicator = engine is running and processing campaigns

8. Settings

The Settings tab lets you configure global defaults:

  • Delay Settings: Min/max delay, typing delay, seen delay between messages
  • Rate Limits: Hourly and daily message caps
  • Safety: Max consecutive failures before auto-pause, engine poll interval
  • Invisible Markers: Toggle the zero-width character feature on/off
  • Warm-Up Schedule: Customize daily limits for each warm-up day
Tip: Campaign-specific settings override the global defaults. Each campaign can have its own delay and limit settings.

9. FAQ & Troubleshooting

Campaign paused automatically?

This happens when 3+ consecutive messages fail. Check if your WhatsApp is still connected (Zoho → WhatsApp tab). Reconnect if needed, then resume the campaign.

Branch dropdown is empty?

Make sure branches are set up in the system (Branches & Staff section). At least one active branch is needed.

No contacts showing?

Only leads with phone numbers appear in the contact browser. Go to the Leads module and make sure your leads have phone numbers saved.

Messages not sending?

Check these things:

  1. Is the engine running? (Green indicator in Dashboard)
  2. Is WhatsApp connected for the campaign's branch?
  3. Have daily/hourly limits been reached?
  4. Is the campaign status "Running"?

How to send to just one person?

Create a campaign, and in Step 3 (Select Contacts), simply select only the one person you want to send to. Campaigns work for both single and bulk messages.

Can I send images?

Yes! In Step 1, set Message Type to "Image". In Step 2, upload your image file and add an optional caption. The image will be sent to each selected contact.
